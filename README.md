# Makeup Match · AI-Powered Recommendations
# Name: Sijie Guo
# Email: sijieg@uchicago.edu
A makeup recommendation system built with **RAG (Retrieval Augmented Generation)**: users upload a face photo → AI analyzes face shape, skin tone, and features → retrieves from a makeup knowledge base → generates personalized makeup recommendations.

## Tech Stack

- **RAG**: ChromaDB + OpenAI `text-embedding-3-small` for vector retrieval over a makeup knowledge base
- **Vision**: OpenAI GPT-4o-mini for face analysis
- **LLM**: OpenAI GPT-4o to synthesize RAG results and generate recommendations
- **Backend**: FastAPI
- **Frontend**: React + Vite + Tailwind

## Quick Start

### 1. Setup

```bash
# Clone / enter project
cd GenAI_final_project

# Set OpenAI API Key (required)
export OPENAI_API_KEY="sk-xxx"
# Or create a .env file in backend/ (loaded via python-dotenv)
```

### 2. Build RAG Knowledge Base (required on first run)

```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python scripts/build_rag.py
```

Chunks, embeds, and stores documents from `data/makeup_knowledge/` into `chroma_db/`.

### 3. Start Backend

```bash
cd backend
source .venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 4. Start Frontend

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000 in your browser and upload a face photo to get recommendations.

## RAG Pipeline

1. **Knowledge base**: Markdown docs on face shapes, skin tones, makeup styles, and features
2. **Chunking**: ~400 chars per chunk with 80-char overlap
3. **Embedding**: OpenAI `text-embedding-3-small` for vectorization
4. **Retrieval**: User photo → Vision extracts face shape/skin tone/eye type → build query → ChromaDB similarity search (top-6)
5. **Generation**: LLM combines retrieval results + face analysis → personalized makeup recommendations

## Responsible AI

- Photos are used for analysis only and are not stored
- Knowledge base covers diverse face shapes and skin tones; prompts emphasize inclusivity
- Recommendations include reasoning traceable to the knowledge base
